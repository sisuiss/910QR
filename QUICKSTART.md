# クイックスタート

## ステップ1：スプレッドシートを準備

1. [Google Sheets](https://sheets.google.com) で新規スプレッドシートを作成
2. シート名を **「Logs」** に変更
3. 1行目にヘッダーを入力：

| A | B | C | D | E | F |
|---|---|---|---|---|---|
| 発行日時 | ブラウザID | トークン | ステータス | 使用日時 | 備考 |

---

## ステップ2：GASを設定

1. スプレッドシートの **拡張機能 → Apps Script** を開く
2. `Code.gs` の内容を全て削除
3. このリポジトリの [Code.gs](Code.gs) の内容を貼り付けて保存

---

## ステップ3：GASをデプロイ

1. GASエディタの右上 **「デプロイ」→「新しいデプロイ」**
2. 種類：**「Webアプリ」**
3. 実行者：**「自分」**
4. アクセス権限：**「全員」**
5. 「デプロイ」をクリック
6. 生成された **WebアプリURL** をコピー

---

## ステップ4：URLを設定

`index.html` と `staff.html` の `GAS_URL` にデプロイしたURLを貼り付ける：

```javascript
const GAS_URL = 'https://script.google.com/macros/s/xxxxx/exec';
```

---

## ステップ5：GitHub Pagesで公開

1. GitHubにプッシュ
2. GitHub Pages を有効にする（Settings → Pages → Source: main）
3. アクセスURL：`https://[ユーザー名].github.io/910QR/index.html`

---

## 動作確認チェック

- [ ] `index.html` を開くとQRが表示される
- [ ] 2回目のアクセスで「お一人様1回限り」エラーが出る
- [ ] スタッフがQRをスキャン → `staff.html` に遷移する
- [ ] パスワード「910」で「承認成功」が表示される
- [ ] ユーザー側が「使用済み」に自動更新される（約3秒以内）
- [ ] Logsシートにデータが記録されている

---

## GASの再デプロイ手順（Code.gsを変更した場合）

1. **「デプロイ」→「デプロイを管理」**
2. 鉛筆マーク（編集）をクリック
3. バージョン：**「新しいバージョン」** を選択
4. 「デプロイ」をクリック
